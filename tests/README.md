# Tests


This folder contains scripts that test the core features of `local-vectors` in different examples.


### Usage

From the root of the repository, run the command `uv run pytest tests/{target_script}.py` to run a specific test script. To run all test scripts, adjust the command as follows: `uv run pytest tests/`. Be sure the uv environment is active before running the script.

You can build the uv environment by running the commands `uv sync` followed by `uv build` from the root of the repository.
