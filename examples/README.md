# Examples


This folder contains scripts that use the core features of `local-vectors` in different examples.


### Usage

From the root of the repository, run the command `uv run python examples/{target_script}.py` to run an example script. Be sure the uv environment is active before running the script.

You can build the uv environment by running the commands `uv sync` followed by `uv build` from the root of the repository.
